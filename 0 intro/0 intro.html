课程介绍

该课程结束后，你将：

  1. 具备使用 HTML / CSS / JavaScript 制作简单静态网页的能力
  2. 了解 Git ，并会使用 Git 的常用命令

  4. 熟悉 Linux 系统下常用命令
  5. 熟悉 Markdown 语法
  6. 掌握自学编程的基本方法
  7. 学会提问技巧
  8. 结识一群热爱编程的小伙伴
